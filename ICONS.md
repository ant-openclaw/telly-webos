# Placeholder Icons

You need to create these PNG images:

- **icon.png** — 128x128 pixels (app icon)
- **icon-large.png** — 512x512 pixels (store/high-res display)
- **splash.png** — 1920x1080 pixels (app launch screen)

You can use the same icon design as the Linux version (blue TV with play button) or create a new one optimized for TV display.

## webOS TV Library

Download from: https://webostv.developer.lge.com/develop/app-developer-guide/setup-webos-tv-sdk/

Extract `webOSTVjs-v1.2.4.zip` to a folder named `webOSTVjs-1.2.4/` in this directory.
